# IT Asset Manager

CRUD de ativos de TI e vulnerabilidades em Python

## Como executar
```bash
python main.py
```

## Requisitos
- Python 3.10+

## Estrutura
- `models/` — entidades do domínio
- `services/` — lógica de negócio e acesso a dados
- `utils/` — validações e helpers
- `data/` — arquivos de persistência em texto
